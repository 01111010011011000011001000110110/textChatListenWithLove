# Spotify Jam Odası

GitHub Pages üzerinde yayınlanabilen, Spotify Jam bağlantısı ve ortak çalma listesi için hazırlanmış özel bir dinleme odasıdır.

## Genel Bakış

Bu proje, Spotify üzerinden birlikte müzik dinleme deneyimini daha düzenli ve estetik hale getirmek için hazırlanmıştır.

Sayfa içinde Spotify çalma listesi bağlantısı, Jam davet bağlantısı, çalma listesi önizleme kartı, canlı mesaj alanı, sesli bağlantı modu ve dinleme seansı sayacı bulunur.

## Özellikler

- Şifreli oda girişi
- Kişi seçimi
- Canlı çevrimiçi durum göstergesi
- Firebase destekli canlı mesajlaşma
- Spotify çalma listesi bağlantısı ekleme
- Spotify Jam davet bağlantısı ekleme
- Spotify çalma listesi önizleme kartı
- Mesajlaşma / sesli bağlantı modları
- İki taraflı mikrofon izinli sesli bağlantı
- Sesli bağlantı süresi, tarihli görüşme geçmişi, askıya alma ve kapatma kontrolleri
- Dinleme seansı sayacı
- Mobil uyumlu arayüz
- GitHub Pages desteği

## Kullanım

1. Siteye giriş yapılır.
2. Kişi seçimi yapılır.
3. Mesajlaşma otomatik aktif olur.
4. Spotify çalma listesi ve Jam bağlantısı eklenir.
5. Jam bağlantısı üzerinden Spotify uygulamasında aynı oturuma katılım sağlanır.
6. Sesli bağlantı için iki taraf da butona basıp mikrofon izni verir.
7. Dinleme seansı başlatılarak ortak zaman takip edilebilir.

## GitHub Pages Yayını

Dosyalar repository ana dizinine yüklendikten sonra GitHub Pages üzerinden yayınlanabilir.

```text
Settings > Pages > Deploy from branch > main > root
```
